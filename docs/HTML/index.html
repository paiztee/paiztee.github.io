HTML

Targeting Elements Ref List

  • by TYPE: $(“button”) - HTML elements are built into HTML.
  • by CSS CLASS: $(".btn") - Classes are NOT unique. You can use the same class on multiple elements. You can use multiple classes on the same element.
  • by CSS ID: $("#target1") - ID’s are unique! Each element can have only one ID. Each page can have only one element with that ID.

ELEMENT encompasses opening tag, closing tag, content. Ex: <p>This is the content</p> This complete thing is called an HTML element.
TAG is just opening or closing entity. Ex: <p> and </p> are called HTML tags.
